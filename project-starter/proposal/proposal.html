1. Introduction

This project uses a soil carbon dataset derived from McNicol et al. (2019), a published study of soil organic carbon stocks in the North Pacific coastal temperate rainforest. The study region spans southeast Alaska and coastal British Columbia, and the original research compiled more than 1,300 soil profiles (pedons) to estimate soil organic carbon stocks to 1 m depth. Soil organic carbon is an important component of the global carbon cycle, so understanding how it varies across soil types can help improve our understanding of carbon storage and ecosystem processes.

In this proposal, I ask whether mean soil organic carbon differs between mineral and organic soils at the pedon level. The response variable is total carbon to 1 m depth (total_c_1m), and the grouping variable is soil type (mineral vs. organic). This question is motivated by the expectation that organic soils typically store more carbon, although initial exploration suggests the pattern may be more complex.

In addition to this primary question, I will also explore how soil carbon varies across different soil orders, and whether there is a relationship between soil carbon and latitude. These additional questions allow for the application of a broader set of statistical methods, including ANOVA and linear regression, and help provide a more comprehensive understanding of the factors influencing soil carbon storage.

Because the dataset is structured at the horizon level (each row represents a soil horizon), multiple observations correspond to a single pedon. Therefore, the analysis is conducted at the pedon level so that each soil profile contributes one independent observation.

3. Data analysis plan

# keep one carbon value per pedon
pedon_carbon <- df %>%
  filter(pedon_start == TRUE) %>%
  select(pedon_id, total_c_1m)

# classify pedons based on whether they contain organic horizons
pedon_type <- df %>%
  select(pedon_id, horizon_type) %>%
  mutate(horizon_type = tolower(horizon_type)) %>%
  group_by(pedon_id) %>%
  summarise(
    pedon_type = if_else(any(horizon_type == "organic", na.rm = TRUE),
                         "organic", "mineral"),
    .groups = "drop"
  )

# combine datasets
final_df <- left_join(pedon_carbon, pedon_type, by = "pedon_id")

# check group sizes
final_df %>%
  count(pedon_type)
## # A tibble: 2 × 2

The data are aggregated so that each pedon contributes one observation, ensuring that the unit of analysis is consistent and that observations are independent.

# exploratory plot
ggplot(final_df, aes(x = pedon_type, y = total_c_1m, fill = pedon_type)) +
  geom_boxplot(alpha = 0.7) +
  geom_jitter(width = 0.1, alpha = 0.4) +
  theme_classic() +
  labs(
    title = "Soil carbon by pedon type",
    x = "Pedon type",
    y = "Total carbon (0–1 m)"
  ) +
  theme(legend.position = "none")

The boxplot compares soil carbon between mineral and organic soils. The individual data points are shown to illustrate the distribution and variability within each group, providing additional context beyond the summary statistics displayed by the boxplot.

Mineral soils appear to have higher median carbon values, although there is substantial variability and overlap between groups. This pattern is somewhat unexpected and suggests that the relationship between soil type and carbon storage may be more complex than initially assumed.
